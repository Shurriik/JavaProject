# Тема проекта "Ресторан"
# Полноценное RESTful веб-приложение для управления бронированием столиков в ресторане.
# Возможности:
 Пользовательская часть
* Просмотр всех доступных столиков
* Поиск столиков по различным критериям (номер, расположение, вместимость)
* Фильтрация свободных столиков
* Создание новых столиков
  
  Административная часть
* Полный управление столиками
* Удаление с возможностью восстановления
* Просмотр удаленных столиков
* Дополнительная информация о столиках

# Инструменты разработки
* VS Code - среда разработки
* Postman - тестирование API
* Git - контроль версий
* Checkstyle - проверка стиля кода
