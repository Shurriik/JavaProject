# Тема проекта "Ресторан"
# Полноценное RESTful веб-приложение для управления бронированием столиков в ресторане.
# Возможности:
 Пользовательская часть
* Просмотр всех доступных столиков
* Поиск столиков по различным критериям (номер, расположение, вместимость)
* Фильтрация свободных столиков

Создание новых столиков
